a tag (link)

문서에서 다른 문서, 다른 파일, 다른 위치로 링크를 만드는 태그

  • a태그는 글자(및 그림)에 링크를 부여한다
  • a태그는 줄변경을 발생시키지 않는다
  • a태크의 대상은 html 문서 혹은 웹 주소, 파일이 될 수 있다
  • a태그의 대상이 현재 문서내부의 다른 ㅇ소가 될 수 있다
  • href : Http Reference, 링크 참조 대상
  • target : 링크가 열릴창, 현재 창 혹은 새 탭등으로 지정 가능
  • a태그의 대상이 브라우저가 처리 가능한 파일이면 바로 열린다
  • a태그의 대상이 브라우저가 처리 불가능한 파일이면 다운로드된다

떡잎마을 유치원 멤버에는 짱구 유리 철수 훈이 맹구 흰둥이 가 있다

외부 파일 링크

브라우저에서 열리는 파일에 우클릭 후 다른 이름으로 저장하면 다운로드 가능

문서내의 특정 요소를 id로 지목하여 페이지 내부 이동을 구현할 수 있다

링크의 대상으로 지정하기 위해 태그에 id를 부여해야 한다

HTML에서 id를 대상으로 지정할 때는 id앞에 # 기호를 붙인다
